# Vuepressでブログを書いてみる。

Vuepressでブログを書いてみる。vuepressとadsenseとSEOとOGPやってみる。

npx create-vuepress-site [optionalDirectoryName]
cd [optionalDirectoryName]
npm install -D vuepress
npm install --save vuepress-plugin-google-adsense 
npm install --save vuepress-plugin-seo

adsenseにadsense-idを追記：

module.exports = {
  plugins: [
    [
      "vuepress-plugin-google-adsense",
      {
        ad_client: "ca-pub-3850197898480094" // replace it with your ad_client
      }
    ]
  ],
